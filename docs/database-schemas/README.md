# Esquemas de Base de Datos

En este directorio se deben almacenar los esquemas y mapeos de datos de las tres aplicaciones:

1. **App 1 (Sari)**: Documentar el esquema JSON de MongoDB (`mongo_schema.json` o un Markdown con la estructura de la colección `osint_raw_data`).
2. **App 2 (Samu)**: Agregar los scripts SQL (`schema.sql` de PostgreSQL) y la descripción de llaves de Redis.
3. **App 3 (Persona 3)**: Agregar los scripts SQL de PostgreSQL (alertas) y el archivo de mapeo JSON de Elasticsearch (`elasticsearch_mapping.json`).
